עורך ניקוד (Standalone)

ציור: opentype.js. ייצוא פונט אמיתי עם MarkToBase דורש שרת Python מקומי (לא ניתן מ־opentype.js לבד).

גרור לכאן .ttf / .otf או לחץ לבחירה

למשל Frank Ruhl Libre. חייב להיות TTF עם glyf (לא CFF בלבד). כשבוחרים כאן, «ייצוא פונט» מפעיל מיזוג: מתארי אותיות עבריות מה־Legacy בתוך ה־Engine, שומר GPOS של ה־Engine, ואז מחיל את ה־JSON.

גרור Engine או לחץ (ריק = ייצוא רגיל על Legacy בלבד)

ריק = שמות מקוריים של ה־Engine. אם ממלאים: משפחה ושם מלא בטבלת השמות, ושם PostScript נגזר (ASCII) כדי שהפונט יופיע נפרד באינדיזיין.
